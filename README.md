# spider-flow

#### 介绍
新一代爬虫平台，以图形化方式定义爬虫流程，不写代码即可完成爬虫。

#### demo
- 妹子图（感谢网友提供）
- 北京菜价（感谢网友提供）
- GVP项目

#### 特性
- [x] 支持css选择器、正则提取
- [x] 支持JSON/XML格式
- [x] 支持Xpath/JsonPath提取
- [x] 支持多数据源、SQL select/insert/update/delete
- [ ] 支持爬取JS动态渲染的页面
- [x] 支持代理
- [x] 支持二进制格式
- [x] 支持保存/读取文件(csv、xls、jpg等)
- [x] 常用字符串、日期、文件、加解密等函数
- [ ] 支持流程嵌套
- [ ] 支持插件扩展(自定义执行器，自定义函数）
- [ ] 任务监控
- [ ] 支持HTTP接口

#### 插件列表
- [ ] Redis插件
- [ ] Mongodb插件
- [ ] Hbase插件
- [ ] IP代理池插件
- [ ] OCR识别插件
